Innosight Model Database Viewer

The Innosight Institute has researched a list of fascinating case studies of schools that have experimented with some form of blended learning. In 2012, they wrote a white paper with findings from about 40 of them. Since then, the list of user submissions has grown.

This website offers a simple visual layer on top of the Innosight data. You can use this to search, filter, and sort by a few major criteria. Click through any of the models to read the much more in-depth coverage on the Innosight website. (This is still pretty basic. Feel free to view or contribute to the source code on Github).

Some of what you can do:

  • Search in the box to the right. You can search for terms that appear anywhere in the model content - for instance, enter "khan" to see all that are using Khan Academy.
  • Filter by clicking on the column titles.
  • View More Columns by clicking "Show/Hide Columns"
